Internet Explorer

Internet Explorer

If you are on a machine which only has this browser - this interface has a significant, but not quite grippling limitation (due to my inability to find how the IE model breaks, in this case, with the standard).

Limitation: All drags and gestures must be inside and to TEXT of the "div" (marked by a box). This means that most mark gestures will be so limited. Simple clicks are unaffected.

Other problems will inevitably crop up (IE is very different from the standard), eg bad fonts, overlapping text etc - i cannot trace down and make allowances for all IE's special needs. So they will just have to remain.

For these reasons it is much easier to use FireFox, Opera, Chrome, Safari etc rather than IE. Here is a link to FireFox.
