A mockup with image rotational capabilities...

Use the UI to reSize and reArrange PICS. [keyboard & pointer]
NB. If the UI loses FOCUS: Clicking on witespace should find it!

[pointer only:]
Click on a left { to show next image in its subgroup as the FIRST image...
DoubleClick on a left { to start rotation of images in its subgroup to the FIRST image...
Click on any right } to end a rotation.

Click on an image to grow. [or to set marker in UI]
DoubleClick on an image to web size it. [alternating web to thumb sizes]
Set dwell time, thumb, and web sizes in boxes.
